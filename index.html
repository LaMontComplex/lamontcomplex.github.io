1. The Groups

Notices group - All residents should consider joining the Notices group - this is
where the Trustees send out important messages. Chat is limited to Administrators
to minimize disturbance.

Alerts - This group is for security and emergency related matters only. Please
do not use this group for anything else.

Community - This group is for general chit-chat, asking questions and is open for
all residents.

The Rules

  1. Respect the purpose of each group. If unsure use a personal message.
  2. Communicate between 06:00 and 21:00 only unless an emergency/security issue.
  3. Do not send content, information or “news” that has not been verified as true.
  4. No messages with bad/foul language or unsavoury pictures will be tolerated.
  5. No messages with political, denominational, religious or racist undertones
    will be tolerated.
  6. No advertising.
  7. Use emoticons sparingly and only reply to messages if adding value or seeking
    more information.
  8. Don’t respond with Thanks and Noted - reading the message is acknowledgment.
  9. If the group(s) are too busy, consider muting the the notifications.
  10. Contact the administrator if unsure or you have questions.

Ask These Three Questions Before Posting

  1. Is this relevant?
  2. Is this necessary?
  3. Is this a good time to post?
